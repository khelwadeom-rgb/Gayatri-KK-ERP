# 🌾 Gayatri Krushi Kendra ERP (GKK ERP)

A modern Agriculture ERP platform designed for Krushi Kendras, Seed Dealers, Fertilizer Shops, Pesticide Retailers, and Agri Business Owners.

The application helps manage daily operations including GST billing, inventory, farmer accounts, purchases, suppliers, udhari management, reports, cloud synchronization, and AI-powered agricultural assistance.
